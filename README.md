# Project-Parkour
This repository contains source and data folders for the game Project Parkour developed for the Trampolima framework.

Keyboard Game controls:

At menu:
* "1" - Easy Level 
* "2" - Hard Level 

Overall controls:
* "w" - High Jump
* "s" - Normal Jump
* "x" - Slide

Developer debug: 
* "z" - Instant Win
